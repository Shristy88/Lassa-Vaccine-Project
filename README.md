# Lassa Vaccine Project: In-Silico Epitope Design

## Overview
This project focuses on the computational identification of safe and immunogenic T-cell and B-cell epitopes for the Lassa Virus Glycoprotein Precursor (GPC). The workflow utilizes immunoinformatics tools to predict, filter, and validate vaccine candidates.

## Methodology
- **Sequence Retrieval:** NCBI Protein Database (RefSeq).
- **Epitope Prediction:** IEDB NetMHCpan 4.1 EL.
- **Safety Profiling:** Allergenicity (AllerTOP) & Toxicity (ToxinPred 3).
- **Structural Validation:** RCSB PDB (5VK2) 3D Mapping.

## Key Findings
- Successfully identified high-priority, non-toxic, and non-allergenic epitopes.
- Validated structural accessibility via 3D molecular modeling.
